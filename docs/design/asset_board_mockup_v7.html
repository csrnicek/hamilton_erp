Hamilton Asset Board — V3

With filter chips and fixed OOS visibility

👆 Try tapping the filter chips at the bottom of the tablet — Dirty, Occupied, OOS, etc.

Final spec

  • Layout: 6 subtype rows with header bars on top
  • Tile colour = status: green = Available, red = Occupied, orange = Dirty, grey-outlined box = Out of Service
  • NewOOS tile style: 2px grey border with lighter background so the tile edges are clearly visible
  • Sort within row: Available → Occupied → Dirty → Out of Service
  • Row order: Single Regular, Single TV, Double Regular, Double TV, VIP, VIP Glory Hole
  • Header bar: subtype name (left) • "X of Y free" count (right)
  • NewFilter chips at bottom: All, Available, Occupied, Dirty, Out of Service — tap one to isolate only those tiles, tap "All" to clear. Empty subtype rows hide automatically when filtered.
